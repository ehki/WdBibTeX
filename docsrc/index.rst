WdBibTeX documentation
======================

WdBibTeX is a BibTeX toolkit for MS Word.


Quick start
-----------

1. Collect ``foo.docx``, ``bar.bib``, and ``baz.bst`` in one directory.

2. Write ``\cite{qux}`` and ``\thebibliography`` to foo.docx.

3. Execute: ``python -m wdbibtex``
